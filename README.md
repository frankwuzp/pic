# 图库说明
个人私有图片库，用于实现markdown图床上传

## img 文件夹
用于平时md写作时图片文件的存放

## blog 文件夹
存放`blog.wuzhiping.top`网站的图片，使用jsDelivr进行CDN加速
详见隔壁仓库👉[jsDelivr加速的说明](https://github.com/frankwuzp/BlogCDN/blob/5941d7ca0d6c929c6f9eb102e872f9ca99f72aa7/README.md)
